# Menu Jeux Complet

Un menu interactif pour accéder facilement à différents jeux HTML/CSS/JS.

## Fonctionnalités

- Navigation dans les catégories de jeux.
- Ouverture des jeux dans de nouvelles fenêtres.
- Bouton "Retour" pour revenir à la catégorie précédente.
- Interface simple et responsive.

## Structure du projet

- `main.html` : page principale avec le menu.
- `main.css` : styles CSS du menu.
- `main.js` : logique JavaScript pour la navigation du menu.
- Les dossiers comme `casino/`, `chess/`, etc. contiennent les jeux.
